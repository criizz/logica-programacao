O que é Lógica de Programação?

Lógica de programação é a técnica utilizada para criar soluções de forma organizada e sequencial, permitindo que computadores executem tarefas corretamente.

Ela é a base para aprender qualquer linguagem de programação.

Algoritmos

Um algoritmo é uma sequência de passos utilizada para resolver um problema ou executar uma tarefa.

1. Ligar a cafeteira
2. Colocar café
3. Adicionar água
4. Preparar café

Variáveis

Variáveis armazenam informações que podem ser utilizadas durante a execução de um programa.

nome = "João" idade = 20

Estruturas Condicionais

As condicionais permitem que o programa tome decisões.

SE idade >= 18 Mostrar "Maior de idade" SENÃO Mostrar "Menor de idade"

Laços de Repetição

Permitem executar uma mesma ação várias vezes.

PARA contador de 1 até 5 Mostrar contador
