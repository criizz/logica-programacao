O que é Lógica de Programação?

Lógica de programação é a forma de organizar pensamentos e instruções para resolver problemas utilizando computadores.

Antes de aprender qualquer linguagem de programação, é importante entender como criar soluções passo a passo.

Algoritmos

Um algoritmo é uma sequência de passos utilizada para resolver um problema.

1. Pegar café
2. Adicionar açúcar
3. Misturar
4. Beber

Variáveis

Variáveis são espaços utilizados para armazenar informações.

nome = "João" idade = 20

Estruturas Condicionais

Permitem que o programa tome decisões.

SE idade >= 18 Mostrar "Maior de idade" SENÃO Mostrar "Menor de idade"

Laços de Repetição

Permitem executar uma tarefa várias vezes.

PARA contador de 1 até 5 Mostrar contador
